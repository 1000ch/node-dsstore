# dsstore ![GitHub Actions Status](https://github.com/1000ch/node-dsstore/workflows/test/badge.svg?branch=master)

Remove `.DS_Store` recursively.

## Install

```bash
$ npm install -g dsstore
```

## Usage

```javascript
import dsstore from 'dsstore';

dsstore(['foo']);
dsstore(['bar', 'baz']);
```
